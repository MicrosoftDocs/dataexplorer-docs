# current_principal_details()

Returns details of the principal running the query.

## Syntax

`current_principal_details()`

## Returns

The details of the current principal as a `dynamic`.

## Example

<!-- csl: https://help.kusto.windows.net/Samples -->
```kusto
print d=current_principal_details()
```
