# isnotempty()

Returns `true` if the argument isn't an empty string, and it isn't null.

```kusto
isnotempty("") == false
```

## Syntax

`isnotempty(`[*value*]`)`

`notempty(`[*value*]`)` -- alias of `isnotempty`
