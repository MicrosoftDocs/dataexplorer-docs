# binary_and()

Returns a result of the bitwise `and` operation between two values.

```kusto
binary_and(x,y)	
```

## Syntax

`binary_and(`*num1*`,` *num2* `)`

## Arguments

* *num1*, *num2*: long numbers.

## Returns

Returns logical AND operation on a pair of numbers: num1 & num2.
