# hourofday()

Returns the integer number representing the hour number of the given date

```kusto
hourofday(datetime(2015-12-14 18:54)) == 18
```

## Syntax

`hourofday(`*a_date*`)`

## Arguments

* `a_date`: A `datetime`.

## Returns

`hour number` of the day (0-23).
