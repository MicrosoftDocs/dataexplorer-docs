# asin()

Returns the angle whose sine is the specified number (the inverse operation of [`sin()`](sinfunction.md)) .

## Syntax

`asin(`*x*`)`

## Arguments

* *x*: A real number in range [-1, 1].

## Returns

* The value of the arc sine of `x`
* `null` if `x` < -1 or `x` > 1
