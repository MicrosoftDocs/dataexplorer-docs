# isascii()

Returns `true` if the argument is a valid ascii string.
    
```kusto
isascii("some string") == true
```

## Syntax

`isascii(`[*value*]`)`

## Returns

Indicates whether the argument is a valid ascii string.

## Example

```kusto
T
| where isascii(fieldName)
| count
```
