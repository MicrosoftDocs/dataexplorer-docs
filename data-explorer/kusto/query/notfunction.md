# not()

Reverses the value of its `bool` argument.

```kusto
not(false) == true
```

## Syntax

`not(`*expr*`)`

## Arguments

* *expr*: A `bool` expression to be reversed.

## Returns

Returns the reversed logical value of its `bool` argument.
